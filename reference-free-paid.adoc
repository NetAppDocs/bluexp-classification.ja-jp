= BlueXPの分類機能の廃止
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
BlueXPの分類は、BlueXPの中核機能として追加料金なしで利用できます。すべてのお客様が利用できるBlueXPのコア機能にBlueXPの分類を含めることで、NetAppの主要機能を使用して、カスタマイズされたデータ管理を利用できるようになります。

一部の機能は、BlueXPコアバージョン1.31以降で廃止され、旧バージョン1.30以前で引き続きサポートされます。
